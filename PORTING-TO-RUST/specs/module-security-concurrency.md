## 1. Security Architecture

### 1.1 Current Go State (Summary)

| Aspect | Go Implementation |
|--------|------------------|
| AuthMode | `"none"` or `"local-token"` -- string enum `AuthMode` |
| Bearer token | Sent as `Authorization: Bearer {token}` header for all `/api/v1/*` routes |
| Loopback exception | `/webhook/forward` path: if webhook enabled AND `isLoopbackRemoteAddr(remoteAddr)` true (127.0.0.1, ::1, localhost) -> allow without token |
| LocalToken | Generated by bootstrap, stored in config file as plain text `*string` (no encryption at rest) |
| NodeToken | Random hex generated by loopernet cloud service, stored in `~/.looper/network.json` (0600 permissions) |
| Join keys | One-time use strings, stored in SQLite (`join_keys` table), consumed atomically via `UPDATE ... WHERE consumed_at IS NULL` |
| gh auth | Looper NEVER stores GitHub tokens; delegates entirely to `gh` CLI's own auth store |
| Unsafe Git env vars | 16 vars stripped from agent process environment |
| Shell execution | Direct `exec.Command` (no shell wrapper), no shell injection risk |
| Config file | No strict permission checking on read (Go default) |
| PID file | Simple text file written via `os.WriteFile`, no permission hardening |
| Disclosure | HTML comment marker `<!-- looper:stamp v=1 -->` stamped on PR/issue/commit bodies |
| Worktree safety | 5 checks: empty path, repo-equal, root-equal, under-root, symlink normalization |
| API bind | Defaults to `127.0.0.1` (loopback) |

```go
func authorizeRequest(r *http.Request, path string, cfg config.Config) error {
    // 1. Loopback exception for webhook forward
    if path == webhookForwardPath && cfg.Webhook.Enabled && isLoopbackRemoteAddr(r.RemoteAddr) {
        return nil
    }
    // 2. AuthMode "none" -> unrestricted
    if cfg.Server.AuthMode != AuthModeLocalToken {
        return nil
    }
    // 3. Empty token means misconfigured
    if cfg.Server.LocalToken == nil || *cfg.Server.LocalToken == "" {
        return apiError{code: authMisconfigured, status: 500}

    if token == *cfg.Server.LocalToken {
        return nil
    }
    // 5. Reject
    return apiError{code: unauthorized, status: 401}
}
```

#### Auth Middleware (axum style)

```rust
pub async fn authorize_request(
    req: &Request<Body>,
    path: &str,
    config: &ServerConfig,
) -> Result<(), AuthError> {
    // 1. Loopback exception for webhook forward
    if path == WEBHOOK_FORWARD_PATH && config.webhook_enabled && is_loopback_remote_addr(req) {
        return Ok(());
    }
    // 2. AuthMode "none" -> unrestricted
    if !matches!(config.auth_mode, AuthMode::LocalToken) {
        return Ok(());
    }
    // 3. Empty token -> 500
    let local_token = config.local_token.as_ref().ok_or(AuthError::Misconfigured)?;
    // 4. Bearer token match
    let bearer = extract_bearer_token(req.headers())?;
    if bearer == local_token.as_str() {
        return Ok(());
    }
    // 5. Reject
    Err(AuthError::Unauthorized)
}

fn is_loopback_remote_addr(req: &Request<Body>) -> bool {
    req.extensions()
        .get::<RemoteAddr>()
        .map(|addr| {
            addr.ip().is_loopback()
        })
        .unwrap_or(false)
}
```

```rust
/// Filter environment variables for agent process.
/// Keeps existing env except unsafe keys, then merges caller-provided env.
pub fn build_agent_env(work_dir: &str, config_env: &HashMap<String, String>, input_env: &HashMap<String, String>) -> Vec<(String, String)> {
    let mut result = Vec::new();
    // Inherit parent env, stripping unsafe keys
    for (key, value) in std::env::vars() {
        if !UNSAFE_AGENT_ENV_KEYS.contains(&key.as_str()) {
            result.push((key, value));
        }
    }
    // Set PWD to work dir
    set_or_override(&mut result, "PWD", work_dir.to_string());
    // Merge config env (lower priority)
    for (k, v) in config_env {
        set_or_override(&mut result, k.clone(), v.clone());
    }
    // Merge input env (highest priority)
    for (k, v) in input_env {
        set_or_override(&mut result, k.clone(), v.clone());
    }
    result
}
```

#### Worktree Path Safety (exact Go logic, ported to Rust)

From `internal/worktreesafety/safety.go`:

```rust
#[derive(Debug, Clone)]
pub struct WorktreeCheckInput {
    pub worktree_path: String,
    pub repo_path: String,
    pub worktree_root: String,
}

/// Validate worktree path safety.
/// Returns Err if path is unsafe.
pub fn validate_worktree(input: &WorktreeCheckInput) -> Result<(), WorktreeSafetyError> {
    let worktree_path = input.worktree_path.trim();
    
    // 1. Empty check
    if worktree_path.is_empty() {
        return Err(WorktreeSafetyError::Empty);
    }
    
    // 2. Must not equal repo path
    if paths_equal(worktree_path, &input.repo_path) {
        return Err(WorktreeSafetyError::EqualsRepoPath(worktree_path.to_string()));
    }
    
    // 3-5. Root-based checks
    let root = input.worktree_root.trim();
    if !root.is_empty() {
        // 3. Must not equal root
        if paths_equal(worktree_path, root) {
            return Err(WorktreeSafetyError::EqualsRoot(worktree_path.to_string()));
        }
        // 4. Must be under root
        if !path_is_within_root(worktree_path, root) {
            return Err(WorktreeSafetyError::NotUnderRoot(worktree_path.to_string(), root.to_string()));
        }
        // 5. Symlink normalization (handled within paths_equal and path_is_within_root)
    }
    Ok(())
}

/// Symlink-aware path normalization
fn normalize_path(path: &str) -> PathBuf {
    normalize_path_depth(path, 0)
}

fn normalize_path_depth(path: &str, depth: u32) -> PathBuf {
    if depth > 255 {
        return PathBuf::from(path);
    }
    let abs = if Path::new(path).is_absolute() {
        PathBuf::from(path)
    } else {
        std::env::current_dir().unwrap_or_default().join(path)
    };
    
    let mut components = Vec::new();
    for component in abs.components() {
        match component {
            Component::Normal(s) => {
                let candidate = components.iter().fold(PathBuf::new(), |acc, c| acc.join(c));
                let candidate = candidate.join(s);
                // Resolve symlinks via std::fs::symlink_metadata
                if let Ok(meta) = std::fs::symlink_metadata(&candidate) {
                    if meta.file_type().is_symlink() {
                        if let Ok(target) = std::fs::read_link(&candidate) {
                            if target.is_absolute() {
                                let resolved = normalize_path_depth(target.to_str().unwrap_or(""), depth + 1);
                                components.clear();
                                for c in resolved.components() {
                                    if let Component::Normal(s) = c {
                                        components.push(s.to_string_lossy().to_string());
                                    }
                                }
                                continue;
                            } else {
                                // Relative symlink: resolve relative to current
                                let parent = components.iter().fold(PathBuf::new(), |acc, c| acc.join(c));
                                let resolved = normalize_path_depth(
                                    parent.join(&target).to_str().unwrap_or(""), 
                                    depth + 1
                                );
                                components.clear();
                                for c in resolved.components() {
                                    if let Component::Normal(s) = c {
                                        components.push(s.to_string_lossy().to_string());
                                    }
                                }
                                continue;
                            }
                        }
                    }
                }
                components.push(s.to_string_lossy().to_string());
            }
            Component::ParentDir => {
                components.pop();
            }
            Component::RootDir => {
                components.clear();
            }
            _ => {}
        }
    }
    components.iter().fold(PathBuf::new(), |acc, c| acc.join(c))
}
```

#### PID File and Single Instance Enforcement

Go uses `syscall.Flock` on `~/.looper/looperd.lock` with content `pid={pid} daemon_id={id} started={timestamp}`.

Rust port (using `fs2` crate for file locking):

```rust
use fs2::FileExt;
use std::fs::{self, File, OpenOptions};
use std::io::Write;
use std::path::PathBuf;

pub struct DaemonLock {
    path: PathBuf,
    file: Option<File>,
}

impl DaemonLock {
    pub fn acquire(daemon_id: &str) -> Result<Self, DaemonLockError> {
        let path = daemon_lock_path();
        fs::create_dir_all(path.parent().unwrap())?;
        
        let file = OpenOptions::new()
            .create(true)
            .read(true)
            .write(true)
            .open(&path)?;
        
        // Try exclusive, non-blocking lock
        file.try_lock_exclusive().map_err(|_| {
            // Read holder info
            let holder = fs::read_to_string(&path).unwrap_or_default();
            DaemonLockError::AlreadyHeld(holder.trim().to_string())
        })?;
        
        // Write holder info
        file.set_len(0)?;
        file.seek(std::io::SeekFrom::Start(0))?;
        writeln!(&file, "pid={} daemon_id={} started={}",
            std::process::id(),
            daemon_id,
            chrono::Utc::now().to_rfc3339())?;
        file.sync_all()?;
        
        Ok(Self { path, file: Some(file) })
    }
    
    pub fn release(&mut self) {
        if let Some(file) = self.file.take() {
            let _ = file.unlock();
            let _ = fs::remove_file(&self.path);
        }
    }
}

impl Drop for DaemonLock {
    fn drop(&mut self) {
        self.release();
    }
}
```

#### Stale PID Handling

Check if the process holding the lock is still alive:
```rust
fn check_stale_lock(path: &Path) -> Result<Option<Pid>, StaleLockError> {
    let content = fs::read_to_string(path)?;
    if let Some(pid_str) = content.lines().next()
        .and_then(|l| l.strip_prefix("pid="))
        .and_then(|s| s.split_whitespace().next())
    {
        if let Ok(pid) = pid_str.parse::<i32>() {
            // Check if process exists
            #[cfg(unix)]
            {
                use nix::sys::signal::{kill, Signal};
                match kill(Pid::from_raw(pid), Signal::SIGNULL) {
                    Ok(()) => return Ok(None), // process still alive
                    Err(nix::Error::ESRCH) => return Ok(Some(pid)), // dead, safe to remove
                    Err(_) => return Ok(None), // permission denied, assume alive
                }
            }
        }
    }
    Ok(None)
}
```

### 1.3 Token Lifecycle

| Token | Generated By | Stored In | Permissions | Sent As | Lifecycle |
|-------|-------------|-----------|-------------|---------|-----------|
| LocalToken | bootstrap | config file (YAML) | 0644 (default, warn if world-readable) | Bearer header | Created once, rotated via re-config |
| NodeToken | loopernet cloud (`Join`) | `~/.looper/network.json` | 0600 | Bearer header | Created on join, persisted for daemon lifetime, lost on leave |
| JoinKey | loopernet admin (`CreateJoinKey`) | SQLite (`join_keys` table) | DB-level | Sent in `JoinRequest` body | One-time use, consumed atomically in transaction, deleted after use |
| AdminToken | env var `LOOPERNET_ADMIN_TOKEN` | Environment variable only | N/A | Bearer header (admin API) | Process lifetime, never written to disk |
| Webhook secret | loopernet cloud (auto-generated) | SQLite (`meta` table) + gh hook config | DB-level | Fetched via API at onboarding | Created on first access, persisted |

### 1.4 Secure Defaults

| Setting | Default | Rationale |
|---------|---------|-----------|
| API bind address | `127.0.0.1` | Safe on loopback, not exposed to network |
| AuthMode | `"none"` | Safe on loopback, no auth overhead for local-only setups |
| Webhook forward loopback check | Enabled | Only accepts localhost connections unless proxy headers present |
| Config file permission warning | Check 0o004 | Warn if world-readable |
| Daemon lock file | `~/.looper/looperd.lock` | Prevents multiple daemon instances |
| Unsafe env strip | 16 Git-specific vars | Prevents Git directory confusion attacks |
| Shell execution | Direct exec | No shell injection vector |
| gh token | Never stored | Delegates to `gh` CLI auth store |

#### Webhook Dispatcher

```rust
use tokio::sync::mpsc;
use std::sync::Arc;
use std::sync::atomic::{AtomicUsize, Ordering};

pub struct WebhookDispatcher {
    queue: mpsc::Sender<WebhookWork>,
    active: Arc<AtomicUsize>,
    cancel_token: tokio_util::sync::CancellationToken,
    worker_handles: Vec<JoinHandle<()>>,
}

impl WebhookDispatcher {
    pub fn new(capacity: usize, num_workers: usize) -> Self {
        let (tx, rx) = mpsc::channel(capacity);
        let rx = Arc::new(tokio::sync::Mutex::new(rx));
        let active = Arc::new(AtomicUsize::new(0));
        let cancel_token = tokio_util::sync::CancellationToken::new();
        let mut handles = Vec::with_capacity(num_workers);
        
        for _ in 0..num_workers {
            let rx = rx.clone();
            let active = active.clone();
            let token = cancel_token.child_token();
            handles.push(tokio::spawn(async move {
                worker_loop(rx, active, token).await;
            }));
        }
        
        Self {
            queue: tx,
            active,
            cancel_token,
            worker_handles: handles,
        }
    }
}

async fn worker_loop(
    rx: Arc<Mutex<mpsc::Receiver<WebhookWork>>>,
    active: Arc<AtomicUsize>,
    cancel: tokio_util::sync::CancellationToken,
) {
    loop {
        let work = {
            let mut rx = rx.lock().await;
            tokio::select! {
                biased;
                _ = cancel.cancelled() => break,
                item = rx.recv() => item,
            }
        };
        
        match work {
            Some(item) => {
                active.fetch_add(1, Ordering::SeqCst);
                // Process work item
                active.fetch_sub(1, Ordering::SeqCst);
            }
            None => break, // channel closed
        }
    }
}
```

### 2.5 Lock Hierarchy and Deadlock Prevention

```
Priority 1 (shortest, no .await):
  Cache locks            :: std::sync::Mutex (discovery cache, execution registry)
  
Priority 2 (medium):
  Registry locks         :: std::sync::Mutex (active execution registry,
                            runner-scoped mutexes for worker/fixer/reviewer)
                            
Priority 3 (longest, may .await):
  State locks            :: tokio::sync::RwLock (network manager state, runtime status)
  
Priority 4 (SQLite level):
  Storage                :: SQLite WAL mode + busy_timeout (connection-level locking)
```

**Deadlock prevention rules:**
1. Never hold `tokio::sync::Mutex` across `.await` when another `tokio::sync::Mutex` is held
2. If multiple locks needed, always acquire in priority order (1 -> 2 -> 3 -> 4)
3. Use `std::sync::Mutex` for priority 1-2 (lock duration < 100us, no `.await`)
4. Network manager state: use `tokio::sync::RwLock`, readers don't block each other
5. Avoid nested `tokio::sync::Mutex::lock().await` calls -- use `std::sync::Mutex` for the outer lock if possible

#### Shutdown Ordering (with CancellationToken propagation)

```rust
use tokio_util::sync::CancellationToken;

pub struct Runtime {
    cancel_token: CancellationToken,
    // Child tokens for each subsystem
    scheduler_token: CancellationToken,
    api_server_token: CancellationToken,
    webhook_token: CancellationToken,
    // ...
}

impl Runtime {
    pub async fn shutdown(&self, timeout: Duration) {
        // 1. Signal all subsystems to stop (non-blocking)
        self.cancel_token.cancel();
        
        // 2. Drain API server (hard timeout: 1s)
        //    - Stop accepting new connections
        //    - Wait for in-flight requests to complete
        tokio::select! {
            _ = self.api_server_graceful_shutdown() => {},
            _ = tokio::time::sleep(timeout) => {
                // Force shutdown
            }
        }
        
        // 3. Stop scheduler (graceful: finish current tick)
        //    - No new ticks
        //    - Wait for current tick to complete
        self.scheduler_token.cancel();
        
        // 4. Kill active agents (SIGTERM -> 5s -> SIGKILL)
        self.execution_registry.kill_all().await;
        
        // 5. Stop webhook forwarder (drain queue, complete in-flight work)
        //    - Stop accepting new deliveries
        //    - Wait for workers to drain queue
        self.webhook_dispatcher.stop().await;
        
        // 6. Stop network heartbeat
        self.network_manager.stop().await;
        
        // 7. Release daemon lock
        self.daemon_lock.release();
        
        // 8. Close storage
        //    - Run PRAGMA wal_checkpoint
        //    - Close SQLite connections
        self.storage.close().await;
    }
}
```

### 2.7 Single Instance Enforcement

```rust
pub struct DaemonLock {
    path: PathBuf,
    file: Option<File>,
}

impl DaemonLock {
    /// Acquire exclusive file lock. Exit immediately on conflict.
    pub fn acquire(config: &Config) -> Result<Self, DaemonLockError> {
        let path = lock_file_path(config);
        
        // Ensure parent directory exists
        if let Some(parent) = path.parent() {
            fs::create_dir_all(parent)?;
        }
        
        let file = OpenOptions::new()
            .create(true)
            .read(true)
            .write(true)
            .open(&path)
            .map_err(|e| DaemonLockError::Io(e))?;
        
        // Exclusive, non-blocking lock
        file.try_lock_exclusive()
            .map_err(|_| {
                // Check for stale lock
                let content = fs::read_to_string(&path).unwrap_or_default();
                if let Some(pid) = extract_pid(&content) {
                    if process_is_dead(pid) {
                        // Stale lock -- caller should retry after cleanup
                        return DaemonLockError::StaleLock(pid, content.trim().to_string());
                    }
                }
                DaemonLockError::AlreadyRunning(content.trim().to_string())
            })?;
        
        // Write identity
        file.set_len(0)?;
        file.seek(SeekFrom::Start(0))?;
        writeln!(file, "pid={} daemon_id={} started={}",
            std::process::id(),
            config.daemon_id(),
            chrono::Utc::now().to_rfc3339()
        )?;
        file.sync_all()?;
        
        Ok(Self { path, file: Some(file) })
    }
}
```

```rust
pub fn read_process_command(pid: i32) -> Result<String, ProcessQueryError> {
    #[cfg(target_os = "macos")]
    {
        // macOS: use proc_pidpath via libproc
        // Or fallback: read /proc/{pid}/command via `ps -p {pid} -o command=`
        let output = std::process::Command::new("ps")
            .args(["-p", &pid.to_string(), "-o", "command="])
            .output()?;
        Ok(String::from_utf8_lossy(&output.stdout).trim().to_string())
    }
    
    #[cfg(target_os = "linux")]
    {
        // Linux: read /proc/{pid}/cmdline
        let cmdline = std::fs::read(format!("/proc/{}/cmdline", pid))?;
        // Null-separated, take first
        let first = cmdline.split(|&b| b == 0)
            .next()
            .unwrap_or_default();
        Ok(String::from_utf8_lossy(first).trim().to_string())
    }
    
    #[cfg(not(any(target_os = "macos", target_os = "linux")))]
    {
        let output = std::process::Command::new("ps")
            .args(["-p", &pid.to_string(), "-o", "command="])
            .output()?;
        Ok(String::from_utf8_lossy(&output.stdout).trim().to_string())
    }
}

pub fn verify_process_identity(pid: i32, expected_command: &str) -> Result<bool, ProcessQueryError> {
    let actual = read_process_command(pid)?;
    let actual_base = std::path::Path::new(&actual)
        .file_stem()
        .and_then(|s| s.to_str())
        .unwrap_or(&actual);
    let expected_base = std::path::Path::new(expected_command)
        .file_stem()
        .and_then(|s| s.to_str())
        .unwrap_or(expected_command);
    Ok(actual_base == expected_base)
}
```

- Token lifecycle: generation, storage, comparison (constant-time), rotation
- Daemon lock: test lock acquired, lock conflict, stale lock detection, lock released
- Disclosure stamp: test stamp applied, stamp stripped, stamp idempotent, stamp pattern matching

### 3.4 Testing Strategy (Concurrency)

- Scheduler wake: verify non-blocking send, verify tick coalescing
- Webhook queue: verify deduplication, capacity limit, worker pool behavior
- ActiveExecutionRegistry: concurrent register/kill, deregistration on drop
- Shutdown sequence: verify ordering, timeout behavior, force-kill escalation
- Deadlock prevention: run with `LOOM` or `tokio::task::yield_now()` stress tests
- Race condition: run with `cargo test -- --test-threads=1 --nocapture` + `LOOM_MAX_PREEMPTIONS=1000000`
- CancellationToken propagation: verify all child tokens fire on root cancel
- SLOC target: ~1500 lines for security module, ~2000 lines for concurrency module
